# Issue #6: 種目選択とUI強化の実装 [完了]

## 概要
「3球 / 4球 / 5球」の選択機能追加と、スコアボードの「×」表示および枠線によるUI改善を行う。

## タスク
- [x] 種目選択の共通状態 (`ballCountProvider`) の実装
- [x] 記録画面: 種目タブの追加とフィルタリング実装
- [x] 記録画面: スコアボードの視覚的改善 (×と枠の追加)
- [x] 入力画面: 表示種目の連動とUI微調整
- [x] グラフ画面: 種目フィルタリングの実装

## 実装内容
- `ballCountProvider` により、記録・入力・グラフの各画面で選択中の種目（ボール数）を同期。
- `RecordScreen`: 1セット（10試行）の表示をグリッド形式に変更し、○だけでなく×（チャコール背景）も視認可能に。
- `InputScreen`: 保存時に現在の種目設定を反映。○×トグルUIの枠線を調整。
- `GraphScreen`: 2段のタブバーを導入し、種目ごとのデータ推移を表示可能に。
