# Issue #3: 記録画面の実装 (スコアボード) [完了]

## 概要
保存された練習記録を100回ブロック（10セット）単位で表示するスコアボードを実装する。

## タスク
- [x] 100回ブロック分割ロジックの統合
- [x] PageView によるブロック切り替え
- [x] 上部統計情報（ブロック別・累計）の表示
- [x] 各セットの行表示 (日付 + ○× × 10 + セット成功率)

## 実装結果
- `RecordScreen` にて `PageView.builder` を使用したブロック表示を実装。
- `StatsCalculator.splitIntoBlocks` により最新の記録から順にブロック表示。
- デザイン指示に基づき、○（ライムグリーン）と×（チャコール）を Courier フォントで整列表示。
- ブロックごとの成功率と累計試行数をヘッダーに表示。
