# Issue #8: HUD/サイバーパンク・リデザイン [完了]

## 概要
ユーザー提供のリファレンス画像に基づき、アプリの全UIを近未来的なHUD（ヘッドアップディスプレイ）スタイルへ刷新した。

## 実装内容
- **デザインシステム**: ダークネイビー背景 (#0B1014)、発光シアン (#00F2FF)、ネオンオレンジ (#FF8A00) を基調とする配色。
- **共通コンポーネント**: 四隅にブラケットを持つ `HudFrame`、発光エフェクトを持つ `NeonButton` を作成。
- **画面別のHUD化**:
  - `InputScreen`: 中央に大きな円形HUDゲージ（成功率）を配置。タクティカルな入力グリッド。
  - `RecordScreen`: セクター表示（100回ブロック）と、サイドボーダー付きのログ行。
  - `GraphScreen`: ホログラフィックな演出を施した分析チャート。
  - `SettingsScreen`: システム構成画面として、よりデジタルな外観へ。
- **品質管理**: `flutter analyze` パス済み。インポートおよび未使用変数の整理完了。
