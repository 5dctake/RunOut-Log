# 実装プラン - 「フレッシュ×爽やか」クリーン・メディカルデザインへの刷新

HUD（サイバーパンク）スタイルを全面的に廃止し、清潔感と信頼感を重視した「フレッシュ×爽やか」なプロフェッショナル・デザインに刷新します。

## デザインコンセプト
- **配色**: スカイブルー (#00A8E8) をメインに、ライトグレー (#F8F9FA) の背景、濃いブルー (#007EA7) をアクセントに使用。
- **トーン**: 医療、ヘルスケア、ビジネスに適した「誠実」「清潔」「信頼」を感じさせるライトテーマ。
- **形状**: HUDの角ばったブラケットやネオンを廃止し、適度な角丸（Rounded Corners）と、柔らかい影（Soft Shadows）を用いたカードUIを採用。

## 提案する変更

### 1. デザインシステムの刷新 (`AppColors`, `AppTheme`)
- **背景色**: 深いネイビーから、明るいライトグレー (#F8F9FA) へ変更。
- **文字色**: 白から、視認性の高いダークグレー/ネイビーへ変更。
- **コンポーネント**: `HudFrame` などを、クリーンな `MedicalCard` や `AppButton` にリプレイス。

### 2. 画面別のデザイン更新
- **HomeScreen**: ダークなグリッド背景を廃止し、明るく開放的な背景へ。
- **RecordScreen**: ログ表示を、整理されたリスト形式（ホワイトカード）に変更。
- **InputScreen**: 円形ゲージを、よりモダンでクリーンな円形インジケーターに更新。
- **GraphScreen**: 透過チャートを、スカイブルーベースの美しい折れ線グラフに更新。

### 3. 多言語対応の維持
- 現在の実装（i18n）を維持し、日本語/英語を切り替え可能にします。初期設定は日本語を優先しつつ、HUD風の用語から一般的な用語（記録、入力など）に調整します。

## 検証プラン
- 全画面がライトテーマとして破綻なく表示されるか確認。
- 指定された3色が適切に配置され、清潔感のあるUIになっているか確認。
- `flutter analyze` で静的解析エラーがないことを確認。
